About ChessReviewer

♟

ChessReviewer v1.0.0

Free Local Game Review & Analysis

ChessReviewer is a web application designed to bring advanced, commercial-grade chess game analysis directly to your local computer for free.

Local Engine: Stockfish (UCI)
Accuracy Formula: CAPS2 win probability
Frontend: Vanilla HTML/JS + Tailwind

Developed as an open alternative to premium chess sites. Check the README for how to import/export PGNs.
